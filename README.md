# tuteacuatro
This is an old unmantained html canvas based on tute card game. It works with a custom made game engine wich draws card layers on demand.

This proyect was created many years ago for learning purposes and it's been unchanged since then.

The cards have been taken from Basqueteur deck - https://commons.wikimedia.org/wiki/File:Baraja_espa%C3%B1ola_completa.png
The used font is Google Font Teachers - https://fonts.google.com/specimen/Teachers
